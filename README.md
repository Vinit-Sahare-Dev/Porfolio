# Vinit Sahare - Portfolio

A modern, responsive developer portfolio built with React, TypeScript, and Tailwind CSS.

ðŸ”— **Live:** [vinitsahare.vercel.app](https://vinitsahare.vercel.app)

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- shadcn/ui

## Features

- Responsive design
- Dark/Light theme
- GitHub activity graph
- Project showcase
- Certifications section
- Contact form

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment

This project is configured for deployment on Vercel, Netlify, or any static hosting.

```bash
npm run build
```

The `dist` folder contains the production-ready files.
